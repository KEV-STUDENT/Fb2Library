# План разработки Fb2Library

## 🎯 Общая цель

Создать полноценное приложение для управления библиотекой книг в формате FB2 с двумя интерфейсами: Blazor (веб) и WPF (десктоп), следуя принципам Clean Architecture.

---

## 📅 Спринты

**Статус:** ✅ Завершено

**Задачи:**
- [x] Создать решение (.sln)
- [x] Создать проекты: Domain, Application, Infrastructure, Persistence, Blazor, WPF
- [x] Настроить зависимости между проектами
- [x] Настроить структуру папок
- [x] Создать архитектурную документацию

**Результат:** Готовая структура Clean Architecture

### Sprint 2: Application Layer

**Статус:** ⏳ Планируется

**Задачи:**
- [ ] Создать DTO: `BookDto`, `BookDetailsDto`, `CreateBookDto`
- [ ] Создать команды: `CreateBookCommand`, `DeleteBookCommand`, `UpdateBookCommand`

- [ ] Реализовать хендлеры для каждой команды/запроса

**Ожидаемый результат:** Готовая бизнес-логика (CRUD + поиск)

**Примерный срок:** 2 дня
